# AI Office UI设计方案 - 实时协作式

**版本**: v2.0
**日期**: 2025-11-15
**设计理念**: 左右分栏实时协作式，AI对话驱动文档生成

---

## 一、设计原则

### 1.1 核心设计理念

**实时协作式工作台**，而非传统的分步向导式:

- 左侧: 数据源 + AI对话交互
- 右侧: 文档实时预览和编辑
- 所见即所得: AI生成立即可视化
- 持续迭代: 通过对话不断优化文档

### 1.2 统一的布局模式

所有文档类型(Word、Excel、PPT)采用统一的三栏式布局:

```
┌────────────────────────────────────────────────────────────────┐
│  顶部导航栏: 项目名称 | 保存状态 | 导出 | 分享 | 设置          │
├────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ┌──────────────┐  ┌──────────────────────────────────────┐   │
│  │              │  │                                        │   │
│  │  左侧面板    │  │         右侧工作区                    │   │
│  │  (300-400px) │  │         (自适应宽度)                  │   │
│  │              │  │                                        │   │
│  │ - 数据源     │  │  - 文档编辑器                         │   │
│  │ - AI交互     │  │  - 实时预览                           │   │
│  │ - 大纲/导航  │  │  - 所见即所得                         │   │
│  │              │  │                                        │   │
│  │              │  │                                        │   │
│  └──────────────┘  └──────────────────────────────────────┘   │
│                                                                  │
└────────────────────────────────────────────────────────────────┘
```

#### 2.3.3 实时协作交互

**AI生成流程**:

1. 用户在左侧输入指令: "请补充更多数据"
2. AI开始生成，右侧出现"生成中"提示框
3. 内容流式输出，逐字显示(打字机效果)
4. 生成完成后，内容高亮，提供"接受/修改/重新生成"选项
5. 用户可以直接编辑，或者继续对话优化

```
用户指令 → AI处理 → 流式输出 → 用户确认/编辑 → 下一个指令
   ↑                                              │
   └──────────────────────────────────────────────┘
              (持续迭代优化)
```

class CollaborationManager {
  constructor() {
    this.ws = new WebSocket("wss://api.deepdive.com/collab");
  }

  onAIUpdate(content) {
    // AI生成的内容推送到客户端
    this.broadcastUpdate({
      type: "ai_generation",
      content: content,
      position: this.cursor,
    });
  }

  onUserEdit(change) {
    // 用户编辑同步到服务器
    this.ws.send({
      type: "user_edit",
      change: change,
    });
  }
}
