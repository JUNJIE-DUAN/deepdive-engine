## 1️⃣ 获取 Tenancy OCID

**步骤**:

1. 登录 https://console.oracle.com
2. 点击右上角菜单 → 管理 → 舱室
3. 找到"根舱室"
4. 复制 OCID 字段的值（长格式，以 `ocid1.tenancy.oc1.` 开头）

**格式**: `ocid1.tenancy.oc1.xxxxxxxxxxxxxxxxxxxxxxxxxx`

```
你的 Tenancy OCID: ocid1.tenancy.oc1..aaaaaaaalp72vq523bbru7qtrnyix6s3aotkgf5q4nhsjzd6vtf6wbcqgdma
_________________________________________________________________
```

---

## 2️⃣ 获取 User OCID

**步骤**:

1. 点击右上角菜单 → 身份和访问管理 → 用户

3. 复制 OCID 字段的值（以 `ocid1.user.oc1.` 开头）

**格式**: `ocid1.user.oc1.xxxxxxxxxxxxxxxxxxxxxxxxxx`

```
你的 User OCID: ocid1.user.oc1..aaaaaaaas7vm3r365jphuvgoxqvdw6l4sdericwhkinevtj5txqxrhh46ffq
_________________________________________________________________
```

## 5️⃣ 获取 API 密钥信息

### 方法 A: 使用现有 API 密钥（简单）

**步骤**:

1. 点击右上角菜单 → 身份和访问管理 → 用户
2. 点击你的用户名
3. 左侧选择"API 密钥"
4. 查看已有的密钥，如果有就不需要新建

```
是否已有 API 密钥?
  [ ] 是，我已有 API 密钥
  [ ] 否，需要新建 API 密钥
```

### 方法 B: 新建 API 密钥（如果没有）

**步骤**:

1. 点击右上角菜单 → 身份和访问管理 → 用户 → 你的用户名
2. 左侧选择"API 密钥"
3. 点击"添加 API 密钥"
4. 选择"生成新密钥对"
5. 点击"下载私钥"，保存文件（例如: `oci_api_key.pem`）
6. 点击"添加"

**获得的 Fingerprint**（长字符串，类似: `a1:b2:c3:d4:e5:f6:...`）：

```
你的 API 密钥 Fingerprint:
_________________________________________________________________
```

**私钥文件的位置**（下载后的路径）：

```
你的私钥文件位置 (例如: /Users/username/.oci/oci_api_key.pem):
_________________________________________________________________
```

## 6️⃣ 验证所有信息

在所有信息填写完后，运行以下命令验证：

```bash
# 验证 OCI CLI 连接
oci iam user get --user-id YOUR_USER_OCID

# 验证舱室访问
oci iam compartment get --compartment-id YOUR_COMPARTMENT_OCID

# 应该都返回成功的 JSON 结果
```
